# セットアップ手順（簡単版）

## 🚀 今すぐやること

### ステップ1: Supabaseでスキーマを実行

1. Supabase Dashboardにアクセス
   - https://supabase.com/dashboard
   - プロジェクトを選択

2. SQL Editorを開く
   - 左メニューから「SQL Editor」をクリック

3. スキーマファイルを実行
   - 「New Query」をクリック
   - `supabase-schema-complete-phase1.sql` の内容をコピー＆ペースト
   - 「Run」をクリック
   - エラーが出ないことを確認

### ステップ2: 開発サーバーを起動

ターミナルで以下を実行：

```bash
cd /Users/shugo/Desktop/office-app
npm run dev
```

### ステップ3: ブラウザで確認

1. ブラウザで `http://localhost:3000` にアクセス
2. ログイン
3. 各機能を試す

## 📂 実行すべきSQLファイル

### 必須
- ✅ `supabase-schema-complete-phase1.sql` - フェーズ1の全機能

### オプション（必要に応じて）
- `supabase-schema-permissions.sql` - 権限管理機能

## ⚠️ 注意点

- SQLファイルは順番に実行してください
- エラーが出た場合は、エラーメッセージを確認してください
- 既存のテーブルがある場合は、エラーが出る可能性があります（その場合は無視してOK）
