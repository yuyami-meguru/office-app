# Discord風システム構成の改善案

## 現在の構造とDiscordの対応関係
- **サーバー（Discord）** = **事務所（現在）** ✅ 実装済み
- **チャンネル（Discord）** = **部署（現在）** ⚠️ 改善の余地あり

## 改善提案

### 1. 中央サイドバーを「部署（チャンネル）リスト」に変更

**現在の問題点：**
- 中央サイドバーが機能メニュー（ダッシュボード、メンバー、タスク...）
- 部署が単なる分類タグとして機能している

**改善案：**
- 中央サイドバーに「部署」をチャンネル風に表示
- 各部署をクリックすると、その部署専用のダッシュボードを表示
- 部署ごとに：
  - メンバー一覧（その部署のメンバーのみ）
  - タスク一覧（その部署のタスクのみ）
  - スケジュール（その部署のイベントのみ）
  - ファイル/共有リソース

**構造：**
```
左サイドバー（事務所） → 中央サイドバー（部署/チャンネル） → メイン（部署の内容）
```

### 2. 部署ごとの独立したダッシュボード

**改善案：**
- 部署を選択すると、その部署専用のページを表示
- 部署ページに：
  - その部署のメンバー一覧
  - その部署のタスク
  - その部署のスケジュール
  - 部署の概要情報

### 3. カテゴリ機能（Discordのカテゴリチャンネル）

**改善案：**
- 部署をカテゴリでグループ化
  - 例：「制作部門」カテゴリ → 「歌い手」「編集」
  - 例：「管理部門」カテゴリ → 「人事」「総務」

### 4. 右サイドバーにメンバーリスト表示

**改善案：**
- Discordのように、右サイドバーに現在の事務所のメンバー一覧を表示
- オンライン/オフライン状態（オプション）
- メンバーをクリックでプロフィール表示

### 5. メインエリアのヘッダー改善

**改善案：**
- 現在選択中の部署名を大きく表示
- パンくずリスト（事務所 > 部署 > ページ）
- 検索バー（部署内の検索）

### 6. 部署のピン留め機能

**改善案：**
- よく使う部署を上に固定表示
- 部署の順序をカスタマイズ可能に
