# フェーズ1機能 クイックスタートガイド

## 実装完了した機能

### ✅ タスク管理の高度化
- **タグ機能**: タスクにタグを追加・削除
- **依存関係**: タスク間の依存関係を管理
- **添付ファイル**: タスクにファイルを添付
- **テンプレート**: テンプレートからタスクを作成

### ✅ お知らせ機能の高度化
- **コメント**: お知らせにコメントを追加
- **メンション**: ユーザーをメンション

### ✅ スケジュール管理の高度化
- **参加者管理**: イベントに参加者を追加、ステータス管理
- **コメント**: イベントにコメントを追加
- **添付ファイル**: イベントにファイルを添付

### ✅ ファイル共有の高度化
- **バージョン管理**: ファイルのバージョン履歴
- **タグ**: ファイルにタグを追加
- **コメント**: ファイルにコメントを追加
- **共有リンク**: ファイルの共有リンクを生成

### ✅ 統合検索機能
- `/search` ページで全機能を横断検索

### ✅ プロジェクト管理機能
- `/projects` ページでプロジェクトを管理
- メンバー管理、タスク紐付け

### ✅ チャット機能
- `/chat` ページでリアルタイムチャット
- メンション機能（@username）

### ✅ ワークフロー/承認機能
- `/workflow` ページで承認申請とワークフロー管理

## 使い方

### タスク管理の高度化機能を使う

1. **タグ管理**
   - タスク管理ページの「タグ管理」ボタンをクリック
   - タグ名と色を設定して作成
   - タスクを展開して「タグ」セクションからタグを追加

2. **依存関係**
   - タスクを展開して「依存関係」セクションから依存タスクを追加
   - タスクAがタスクBに依存する場合、タスクBが完了するまでタスクAを開始できない

3. **添付ファイル**
   - タスクを展開して「添付ファイル」セクションからファイルをアップロード
   - 最大50MBまで

4. **テンプレート**
   - 新規タスク追加時に「テンプレートから」を選択
   - テンプレートからタスクを作成（テンプレート機能のUIは今後追加予定）

### 統合検索を使う

1. サイドバーの「検索」をクリック
2. 検索キーワードを入力して検索
3. メンバー、タスク、お知らせ、イベント、ファイルを横断検索

### プロジェクト管理を使う

1. サイドバーの「プロジェクト」をクリック
2. 「+ 新規プロジェクト」からプロジェクトを作成
3. メンバーを追加、タスクを紐付け

### チャットを使う

1. サイドバーの「チャット」をクリック
2. 「+ 新規」からチャットルームを作成
3. メッセージを送信（@usernameでメンション）

### 承認ワークフローを使う

1. サイドバーの「承認」をクリック
2. 「+ 新規ワークフロー」からワークフローを作成
3. 「+ 新規申請」から承認申請を作成
4. 承認・却下・差し戻しを実行

## 注意事項

- 各機能はSupabaseでスキーマを実行済みであることが前提です
- エラーが発生した場合はブラウザのコンソール（F12）を確認してください
- RLSポリシーは `USING (true)` に設定されているため、認証済みユーザーは全データにアクセス可能です
